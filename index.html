Setup VPN

Getting things going
  • Create a IKEv2 VPN with settings:
  • Server: vpn.schmidbauer.cz
  • Remote ID: vpn.schmidbauer.cz
  • Local ID: your@email.com
  • Authentication: none
  • Certificate: Select installed certificate with your@email.com
